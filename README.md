# 🎬 Netflix Content Overview Dashboard

An interactive Tableau dashboard built using the Netflix Titles dataset. This project analyzes Netflix content through various visualizations and provides insights into content distribution, countries, ratings, genres, and release trends.

---

## 📌 Tools Used
- Tableau
- CSV Dataset
- Data Visualization

---

## 📊 Dashboard Features

- Movies vs TV Shows Distribution
- Top 10 Countries by Number of Titles
- Content Released Over Time
- Top Ratings Distribution
- Top Genres Analysis

---

## 🛠 Skills Demonstrated

- Tableau Dashboard Creation
- Data Visualization
- Data Analysis
- Trend Analysis
- Interactive Reporting
